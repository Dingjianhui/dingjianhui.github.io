安装文档-以官方文档为准

官方文档
https://docs.docker.com/engine/install/centos/

Step 1 卸载旧版本

$ sudo yum remove docker \
                  docker-client \
                  docker-client-latest \
                  docker-common \
                  docker-latest \
                  docker-latest-logrotate \
                  docker-logrotate \
                  docker-engine

Step 2 设置存储库

安装utils工具包 （依赖）
官方源信息
yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo
阿里云软件源
yum-config-manager --add-repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

$ sudo yum install -y yum-utils
$ sudo yum-config-manager \
    --add-repo \
    https://download.docker.com/linux/centos/docker-ce.repo

Step 3 安装 docker 引擎

1.安装最新版本
$ sudo yum install docker-ce docker-ce-cli containerd.io
2.指定版本安装
列出存储库中的版本，由高到低

$ yum list docker-ce --showduplicates | sort -r
 * updates: ftp.iij.ad.jp
Loading mirror speeds from cached hostfile
Loaded plugins: fastestmirror
Installed Packages
 * extras: ftp.iij.ad.jp
docker-ce.x86_64            3:19.03.9-3.el7                    docker-ce-stable 
docker-ce.x86_64            3:18.09.9-3.el7                    docker-ce-stable 
docker-ce.x86_64            17.12.1.ce-1.el7.centos            docker-ce-stable

安装 
example : 
    sudo yum install docker-ce-3:19.03.9-3.el7 docker-ce-cli-3:19.03.9-3.el7 containerd.io

$ sudo yum install docker-ce-<VERSION_STRING> docker-ce-cli-<VERSION_STRING> containerd.io

Step 4 启动 docker

systemctl start docker 启动docker
systemctl enable docker 开机启动

$ sudo systemctl start docker
$ sudo systemctl enable docker

Step 5 修改用户组

将普通用户添加至docker组(不然没有权限)
usermod -aG docker ( 这里替换成你自己的用户名)
注意：光加入还不行，要么重新登录
要么执行newgrp - docker 改变当前用户的有效群组

$ sudo usermod -aG docker tintin

安装docker-compose

安装文档： https://docs.docker.com/compose/install/
curl下载失败，可从github上手动下载
https://github.com/docker/compose/releases
1. 下载
$ sudo curl -L "https://github.com/docker/compose/releases/download/1.26.0/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose

要安装其他版本的Compose，使用的Compose版本替换1.26.0

2. 添加可执行权限
$ sudo chmod +x /usr/local/bin/docker-compose
3. 验证
$ docker-compose version
docker-compose version 1.26.0, build d4451659
docker-py version: 4.2.1
CPython version: 3.7.7
OpenSSL version: OpenSSL 1.1.0l  10 Sep 2019
4. 卸载
$ sudo rm /usr/local/bin/docker-compose

     

PS : 阿里云 Docker 镜像加速
阿里云->产品->容器镜像服务->登录控制台
镜像加速器->操作文档 
加速度器地址 需要替换

$ sudo mkdir -p /etc/docker
$ sudo tee /etc/docker/daemon.json <<-'EOF'
{
  "registry-mirrors": ["https://**********.mirror.aliyuncs.com"]
}
EOF
$ sudo systemctl daemon-reload
$ sudo systemctl restart docker
