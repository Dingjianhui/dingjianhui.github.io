windows 开发环境下 go 交叉编译

2020-06-05
golang
约 96 字 预计阅读 1 分钟

文章目录

  • linux
  • windows

linux

set GOOS=linux
set GOARCH=amd64
go build -o build/hugo main.go

windows

set GOOS=windows
set GOARCH=amd64
go build -o build/hugo.exe main.go
